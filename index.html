    Oracle Code Innovate

     

    Oracle Virtual Code Innovate

    Virtual Code Innovate is a 5 day co-development event designed to educate developers on Oracle Cloud while producing a first pass prototype on customer selected business challenge. Over the course of a week 3-4 participants are paired with an Oracle engineer for ~4hrs a day. During this time they will ideate on the business challenge, write a goal statement, identify tasks to complete and partner to develop first pass prototype/minimum viable product (MVP). The event conclude with a 10 minute demonstration of the solution developed. Whether you are a novice or pro in the Cloud, Code Innovate can help jumpstart your next project.
